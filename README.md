# PlanMate

Aplikasi perencana kegiatan kuliah untuk mahasiswa, membantu mencatat jadwal, tugas, dan pengingat secara mudah.

# Deskripsi

PlanMate adalah aplikasi mobile sederhana yang dirancang untuk mahasiswa agar tidak melewatkan jadwal kuliah atau tenggat tugas.
Desain dibuat menggunakan Figma, dengan konsep tampilan lembut dan mudah digunakan.
Deskripsi

# Fitur Utama

# Splash Screen:
Menampilkan logo aplikasi, bendera Indonesia, dan sapaan otomatis sesuai lokasi pengguna.

# Home:
Menampilkan daftar jadwal kuliah serta tombol untuk menambah jadwal baru.

# Tambah Jadwal:
Form input Nama Mata Kuliah, Hari, dan Waktu untuk menambahkan kegiatan baru.

# Pengingat:
Daftar tugas dan ujian dengan opsi menambahkan pengingat baru.

# Pengaturan:
Menu Akun, Notifikasi, dan Tentang Aplikasi.
